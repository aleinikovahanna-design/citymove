# CityMove

CityMove is a smart mobile application that provides real-time public transport tracking,
multi-modal route planning, and live notifications to improve urban mobility.

## Features
- Real-time transport map
- Multi-modal trip planner
- Live notifications
- Training mode
- Admin dashboard

## Tech Stack
- Frontend: React Native
- Backend: Node.js / FastAPI
- Database: PostgreSQL + PostGIS
- Real-time: WebSockets
- CI/CD: GitHub Actions
